    functions

    lighten

    @function lighten() { ... }

    Description

    Función para aclarar un color.

    La función lighten mezcla el color blanco (#ffffff) con el color especificado utilizando la función mix() de Sass. El resultado es un color más claro.

    Parámetros:

    • $color: El color base que se desea aclarar.
    • $amount: La cantidad de aclarado, representada como un número entre 0 y 100.

    Retorna:

    • El color resultante después de aplicar el aclarado.

    Ejemplo de uso:

    $color-aclarado: lighten(#ff0000, 20%);

    Parameters

    None.

    font-familia-titulo

    @function font-familia-titulo() { ... }

    Description

    Función para obtener la familia de fuentes del título.

    La función font-familia-titulo devuelve la cadena de texto que representa la familia de fuentes utilizada para los títulos. En este caso, se devuelve "Arial, sans-serif".

    Retorna:

    • La familia de fuentes para los títulos.

    Ejemplo de uso:

    $familia-titulo: font-familia-titulo();

    Parameters

    None.

    mixins

    tabla-estilos

    @mixin tabla-estilos() { ... }

    Description

    Mixin para los estilos de la tabla.

    El mixin tabla-estilos define los estilos generales de una tabla, incluyendo el borde y el color de fondo. También establece los estilos de las celdas, incluyendo el borde y el relleno, y los estilos de las cabeceras de columna, incluyendo el color de fondo y el color del texto.

    Ejemplo de uso:

    .mi-tabla {
  @include tabla-estilos;
}

    Parameters

    None.

    Requires

    • [variable] color-borde
    • [variable] color-borde
